# Optimisation de l'algorithme de Dijkstra
## Contexte
Projet réalisé en L3 informatique (2025-2026). Parallélisation d'un code C et test de différentes approches pour voir laquelle est la plus efficace.
- université : URCA - UFR Sciences Exactes et Naturelles
- professeur référent : STEFFENEL Luiz-Angelo
- identifiant de la matière : info0604 - Introduction au Calcul Haute Performance

Un compte-rendu du projet décrivant les différentes approches et partis pris ainsi que les analyses correspondantes est présent dans le dépôt.
## Utilisation
__Prérequis__
- Compilateur C (GCC)
- Make
__Compilation et exécution__
```
make
time make run
time make run_seq
```
